# Pho Bamboo 点餐系统 · 使用说明（老板版）

> 这份说明用大白话写，照着做就行。遇到打不开、报错等技术问题，找技术人员；

## ⭐ 每天这样用（看这一段就够了）

平板桌面上那个**「Comptoir」图标**就是柜台，点开它就行（已经登录好，一直开着）。

**一天就重复这 3 件事：**

### 1️⃣ 来单 → 自动响
顾客扫码点餐后，平板会**「叮」一声并弹出新订单**。
卡片上看到：**大数字 = 桌号**（写「À emporter」=外带）、点了什么菜。

### 2️⃣ 出餐 → 点铃铛 🔔
菜做好了，点订单上的**铃铛**，这单变绿色「PRÊT（就绪）」，提醒可以上菜。
（如果你不用这个步骤也行，跳过直接到收款。）

### 3️⃣ 收钱 → 点「ENCAISSÉ」→ 选现金/刷卡
顾客来付款，点订单上蓝色大按钮 **ENCAISSÉ（收款）** → 跳出 **Espèces（现金）/ Carte（刷卡）** 选一个。
✅ 搞定！这单消失，**如果是会员，积分自动加好**。

> 就这么简单。**90% 的时间你只用做第 3 步「收款」。**

**另外常用的两下：**
- **加/查会员积分**：上方切到 **Fidélité（会员）** → 输手机号 → 看积分、可手动加减。
- **看今天营业额**：上方点 **Stats** → 营业额、现金/刷卡、畅销菜一目了然。

下面是更详细的说明，需要时再翻。

## 一、这套系统能做什么？

顾客扫桌上的二维码 → 手机上看菜单、点餐 → 订单自动跳到**柜台平板** →
顾客吃完到柜台付款 → 你点一下「已收款」→ **会员积分自动加好**。

整套系统有 **5 个页面**，各管一摊：

| 页面 | 谁用 | 在什么设备上用 | 作用 |
|---|---|---|---|
| 🍜 **菜单首页** | 顾客 | 手机 | 看菜（不下单） |
| 🛒 **点餐页** | 顾客 | 手机（扫码进入） | 点餐、会员登录、看积分 |
| 💰 **柜台** | 你/店员 | 平板 | 看订单、收款、改积分 |
| 📊 **统计** | 你 | 平板/电脑 | 看营业额、会员、设置 |
| 📝 **菜单管理** | 你 | 手机/电脑 | 加菜、改价、传图 |
| 🔥 **厨房屏**（可选） | 后厨 | 平板 | 实时看待做的菜 |

## 二·五、把它装到主屏，像 App 一样用（推荐）

手机/平板浏览器里打开对应页面 → 浏览器菜单选「**添加到主屏幕**」，就会生成一个图标，点开像独立 App：

- 在 **柜台页**（`/admin/counter`）添加 → 图标直达**柜台**（平板放这个）。
- 在 **点餐页**（`/order`）添加 → 图标直达**点餐**。
- 在 **首页**添加 → 图标是顾客菜单首页。

> - 平板装柜台 App 后，进去用**顶部图标**就能切到 统计 / 菜单 / 二维码，不用退出。
> - 如果装上后打开是登录页，**在 App 里重新登录一次**即可（之后会记住）。
> - 还支持**离线看菜单**（网断了菜单也在，下单需要联网）。

## 三、顾客怎么点餐（你了解即可）

1. 顾客扫**桌上的二维码**，手机自动打开点餐页（已带好桌号）。
2. 选菜，点「**Ajouter / 加入**」放进购物车。
3. 想攒积分就用**手机号登录**（也可以不登录直接点）。
4. 点「**下单并发送**」→ 订单立刻出现在你的**柜台**上。
5. 顾客**吃完到柜台付钱**（系统不收钱，钱还是在柜台收）。

> 顾客右上角有**国旗**可切换 中文 / 英文 / 法文。

## 四、柜台日常操作（最常用！）

打开 `你的网址/admin/login` 登录后，进入**柜台**页面。有两个标签页：

### ▶ 「Commandes / 订单」标签
- 新订单会**自动弹出**（有提示音），大字显示**桌号**（外带显示「À emporter」），上方小标签是当天**订单号 N°**，下面是下单时间和已等待分钟（超 15 分钟变红）。
- 右上角有**🔄 刷新**按钮（万一漏单时点一下重新拉取）和**喇叭**（开关提示音）。
- 每单可点**🖨 打印**小票。
- 菜做好了（如果不用厨房屏）：点订单上的**小铃铛**，状态变「PRÊT 已就绪」。
- **顾客来付款时**：点蓝色的「**ENCAISSÉ / 已收款**」按钮 →
  - 弹出确认框，点确认；
  - 订单从屏幕消失；
  - **如果顾客是会员，积分自动加好**（按钮上会显示大概加多少分）。
- 点错单要取消：点订单右上角的 **✕**（会让你再确认一次）。

**收款时选支付方式**：点「ENCAISSÉ / 已收款」后，会弹出**「Espèces 现金 / Carte 刷卡」**两个大按钮，点对应的那个即可。系统会记录，方便日后对账。

### ▶ 「Fidélité / 会员」标签 —— 手动查/改积分 + 退款
1. 输入顾客**手机号**，点搜索。
2. 看到他当前积分、**姓名**（可填）、**到店次数、累计消费、最近订单**。
3. 快捷按钮：**+100 / +200 / −200 / −500**，或手动输入任意增减。
   （比如顾客没用手机下单，但要补积分，就在这里加。）
4. **退款**：在「最近订单」里点某单后面的**↩ 退款**按钮 → 确认后，该单作废，
   对应的积分自动撤销/退回。

## 五、改菜单（加菜、改价、换图、上下架、删除）

进入**菜单管理**页面（手机或电脑都行）。分「**Plats 菜品**」和
「**Récompenses 奖励**」两个标签。

### 加一道新菜
1. 点右上角「**Ajouter / 添加**」。
2. 填写：
   - **图片**：点「选择」上传一张（手机里的照片就行，系统会自动压缩）。
   - **三种名称**：法语 🇫🇷 / 英语 🇬🇧 / 中文 🇨🇳（不填的语言会自动用法语显示）。
   - **价格、分类**（主菜/前菜/饮料…）、**子分类**（可选，用于分组）、描述。
   - 勾选 **热门 / 辣 / 素 / 上架**。
3. 点「**Enregistrer / 保存**」。

### 改价格 / 改名 / 换图
- 点这道菜上的**铅笔图标** → 改完保存。

### 临时下架（卖完了）
- 点菜右边的**绿色开关**，关掉就不在顾客菜单显示了（不会删除）。随时可再打开。

### 删除一道菜
- 点菜右边的**垃圾桶图标** → 会弹确认框，确认后才真正删除（**删了无法恢复**）。

### 生成桌号二维码
后台顶部 **QR** 页面：填「从第几桌到第几桌」→「Générer 生成」→「Imprimer tout 全部打印」。
打印出来剪开贴到各桌即可，顾客一扫就进入对应桌号的点餐页。也可单独下载某个 PNG。

### 奖励（积分兑换的礼品）
- 切到「Récompenses」标签，同样可以加/改/删，设置**需要多少积分**兑换。

> ⚠️ 从老系统迁移过来的菜，**只有法语名**。有空在这里把中文、英文名补上就行；
> 不补也能用，顾客切语言时会自动显示法语。

## 六、看营业统计

进入**统计**页面（平板/电脑）：
- 今日/累计**营业额**、已付**订单数**、**会员数**、流通**积分**。
- **现金 / 刷卡**营收拆分。
- **每日营收柱状图**（可选 7 / 14 / 30 天）。
- **最畅销菜品**排行。
- **会员排行**（积分最高的顾客）。
- 「**Exporter CSV**」一键导出订单明细，用 Excel 打开做账。
- 右上角「**🔄 Rafraîchir 刷新**」更新到最新数据（统计是进页面时的快照，看最新就点它）。
- （老板账号）底部还有**设置**和**短信**面板。

### 给会员发促销短信（老板，手机操作）
统计页最底部「SMS aux clients opt-in」：列出所有**同意接收促销短信**的会员。
1. 在文本框写好促销内容。
2. 点某个会员后面的「**SMS**」→ 直接打开你手机自带的短信 App，号码和内容都已填好，发送即可。
3. 或点「**Copier les numéros 复制号码**」，把号码粘到短信 App 里建个群发。
> 不需要任何短信网关/付费服务，用你自己手机发，简单可控。只有勾选过"同意接收短信"的顾客才会出现在这里。

## 七、厨房屏（KDS）—— 可选功能，默认关闭

如果后厨想要一块屏幕**实时显示待做的菜**（来新单会响铃），可以打开它：

1. 进入**统计**页面，拉到最下面「**Paramètres / 设置**」（只有老板账号能看到）。
2. 打开「**Module Cuisine (KDS) / 厨房模块**」开关，保存。
3. 顶部菜单会多出「**Cuisine / 厨房**」入口，在后厨平板上打开即可。
4. 后厨做好一道单，点「**TERMINÉ / 完成**」，这单就从厨房屏消失、在柜台变「已就绪」。

不需要就一直关着，不影响其它功能。

## 八、会员积分规则（怎么算的）

- **1 欧元 = 1 分**。
- **当天消费满 20 欧才开始有分**（不满 20 欧不给分）。
- 同一个会员**当天午餐 + 晚餐会累加**：比如中午花 15 欧（暂无分），
  晚上又花 10 欧，当天共 25 欧 → 一共给 25 分。
- 积分在顾客**付款时**由柜台「已收款」自动计入。

> 「满 20 欧」和「1 欧=1 分」这两个数字**可以改**：统计页 → 设置里调整。

## 十一、常见问题

**Q：顾客说扫码点不了餐？**
先确认网络正常；让顾客**刷新页面**；还不行就用柜台手动记一下，事后排查。

**Q：订单没跳到柜台？**
柜台页面**刷新一下**；确认平板联网。

**Q：顾客没登录但要积分？**
柜台「会员」标签，输入他手机号手动加分。

**Q：菜的图片传不上去？**
换张小一点的图、或换个网络重试。

**Q：改了菜单顾客没看到更新？**
让顾客**刷新**手机页面即可。
